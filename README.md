# E-mail Template Generator

Simple and usefull web app to generate e-mails for any reason you want!

## Motivation
Through a study case at Equals, I've built this tool to help the entire company to produce good copy with brand guidelines in a simple way. You can check the product case on medium: [click here](https://stefanobg.com).
